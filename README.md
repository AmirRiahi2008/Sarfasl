# سیلابس کامل دوره CIW Web Foundations
### HTML + CSS + Bootstrap + JavaScript + jQuery — پوشش کامل سرفصل‌های w3schools

> روش تدریس: **تلفیقی (Integrated)** — هر مبحث HTML بلافاصله با استایل‌دهی CSS مرتبطش در همون جلسه تدریس میشه. بعد از تسلط کامل روی HTML/CSS، وارد Bootstrap، JavaScript و jQuery می‌شیم. هر فاز با یک یا چند پروژه عملی بسته می‌شه.

## فاز صفر — Kickoff و راه‌اندازی

**جلسه ۱**
- معرفی مسیر یادگیری و اینکه هر تکنولوژی چه نقشی داره
- نصب VS Code + Live Server + Prettier
- آشنایی با DevTools مرورگر
- ساختار پایه سند: `<!DOCTYPE>`, `<html>`, `<head>`, `<body>`
- Elements، Attributes، Comments در HTML
- اولین فایل CSS و اتصال با `<link>` (External CSS) + Internal/Inline CSS
- 🎯 **پروژه:** اولین صفحه زنده با HTML+CSS
